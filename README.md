# Schautrack Android

The official Android app for [Schautrack](https://schautrack.schauer.to) - an open-source, AI-powered calorie tracker.

## Download

- [Google Play (Beta)](https://play.google.com/apps/testing/to.schauer.schautrack)

## About

This is a native Android wrapper for the Schautrack web application. It provides a seamless mobile experience with native features like camera access for food scanning and push notifications.

## Related Projects

- [Schautrack Web App](https://schautrack.schauer.to)

## Building

```bash
cd android
./gradlew assembleProdDebug
```
